Welcome to the 'My Bog App'. You can use it to create and maintain your favorite creatures like:
* creature name
* description
* an image for each creature

## Technologies Used
* bootstrap for styling the HTML page
* Ruby on Rails
* Active Records for PostgreSQL DB
